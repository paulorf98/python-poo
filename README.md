# Sistema Universitário em Python

Projeto de estudo desenvolvido para praticar Programação Orientada a Objetos (POO) em Python.

## Conceitos praticados

- Classes e objetos
- Herança
- Encapsulamento
- Métodos abstratos (ABC)
- Polimorfismo
- Type hints
- Organização de projetos em múltiplos arquivos

├── __main__.py
