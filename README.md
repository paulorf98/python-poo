# Sistema Universitário em Python

Projeto de estudo desenvolvido para praticar Programação Orientada a Objetos (POO) em Python.

## Conceitos praticados

- imports e pacotes python
- Classes e objetos
- Herança
- Encapsulamento
- Métodos abstratos (ABC)
- Polimorfismo
- Type hints
- Organização de projetos em múltiplos arquivos

## Observações

O projeto foi refatorado para utilizar uma estrutura de pacotes Python com imports absolutos, imports relativos e uso de `__init__.py` como interface do módulo.

## Estrutura

```text
system_university/
├── __main__.py
└── models/
    ├── __init__.py
    ├── pessoa.py
    ├── estudante.py
    ├── professor.py
    └── funcionario.py
```
